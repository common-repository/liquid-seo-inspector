== Description ==

Easily inspect metadata within web pages to visualize and find issues.
WordPress metadata may generate different or duplicate tags than expected, but the issue is quickly discovered.

This checklist will help you address SEO and Core Web Vitals issues.
This tool is designed to check the generated metadata, not to generate it.

Only enabled for administrators and editors who are logged in.

= Features =

* You can easily copy meta tag values.
* Displays the number of characters in the Title, Description, and Headings.
* You can check page load times.
* You can enable / disable on the administration screen.
* Latest information on [LIQUID PRESS](https://lqd.jp/wp/).

== Installation ==

1. Upload 'liquid-seo-inspector' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
